Welcome to FLASHCARD.

This project lets you create, update, and manage flashcards for studying.

  • Click the ▼ icon on the top-right to create a flashcard.
  • Use the search bar above to find specific cards.
  • You can edit, delete, import, or export cards easily.
Need help creating data?

Example Flashcard JSON Structure : 

[
  {
    "title": "Japanese Kanji",
    "fields": [
      { "content": "水", "explanation": "Water" },
      { "content": "火", "explanation": "Fire" },
    ]
  }
]

Each flashcard object must contain a title and a fields array with content and explanation keys.
